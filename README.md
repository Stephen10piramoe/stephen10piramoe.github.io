Piramoe — Developer Portfolio

Local preview

1. Start a simple HTTP server from this folder (Python 3):

```bash
python -m http.server 8000
```

2. Open http://localhost:8000/piramoe-portfolio.html in your browser (or `index.html`).

Deploy options

- GitHub Pages:
  1. Initialize a repo and push to GitHub.
  2. In repo settings enable Pages from the `main` branch (or `gh-pages`).
  3. Your site will be served at `https://<username>.github.io/<repo>`.

- Netlify:
  - Drag-and-drop the `piramoe-portfolio.html` file into Netlify Drop, or connect a GitHub repo and set the build to use the repository root.

- Vercel:
  - Import the GitHub repo in Vercel and deploy.

Notes

- Update the LinkedIn URL in `piramoe-portfolio.html` to your profile.
- To publish, create a GitHub repository and follow the platform instructions to push and enable Pages.
